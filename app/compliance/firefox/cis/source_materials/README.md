# CIS Source Materials

Place official CIS PDFs used for local curation in this directory.

PDF files in this directory are intentionally ignored by Git. Commit derived YAML metadata and mappings instead, unless the licensing boundary is explicitly revisited.

CIS states that non-member PDF versions of CIS Benchmarks are licensed under Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International. That is a non-commercial content license, not the repository's MPL-2.0 code license.
