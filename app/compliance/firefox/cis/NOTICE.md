# CIS Firefox Benchmark Notice

This directory contains project-maintained metadata and mappings derived from the CIS Mozilla Firefox ESR GPO Benchmark.

## Source License Boundary

CIS states that PDF versions of CIS Benchmarks made available as Non-Member CIS Products are provided under the Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International Public License.

That license allows non-commercial sharing and adaptation when its attribution and ShareAlike conditions are met. It is not the same as this repository's MPL-2.0 license, and the NonCommercial restriction means the benchmark material should not be treated as OSI-open-source project code.

The official CIS PDF used for local curation should remain uncommitted unless the project explicitly revisits redistribution and licensing. Derived benchmark metadata should retain source attribution and should not claim CIS certification, CIS endorsement, or a particular level of CIS compliance.
